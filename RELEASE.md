# ModusToolbox&trade; Power Conversion Middleware 1.0

## What's Included?

This is a first production Power Conversion Middleware release.

Refer to the [README.md](./README.md) and the [API Reference Guide](https://infineon.github.io/mtb-pwrconv/html/index.html) for a complete description of the ModusToolbox&trade; Power Conversion Solution.

## Known Issues

* The migration from Middleware version 0.2 (preliminary release) to version 1.0 (the current release) is not supported by ModusToolbox&trade; 3.3 - see the ModusToolbox&trade; 3.3 release notes for details.
